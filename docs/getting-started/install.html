    • Create New Project

    Create project directory

    $ mkdir starter

    NPM Init

    $ cd starter
$ npm init

    Install NamedFramework

    $ npm install --save namedframework

    Create app.js

    // require Framework
const Framework = require('namedframework');

// start Framework
Framework.init();

    Start

    $ nodemon app.js
